# vue.js
学习vue.js

//vue.js环境搭建：
$ npm -v        //查看npm是否安装
$ node -v       //查看node是否安装
$ npm install -g cnpm --registry=https://registry.npm.taobao.org   
		//安装淘宝cnpm镜像文件
$ cnpm -v
$ cnpm install -g(全局安装) vue-cli  //安装vue

$ vue init webpack my-profect(项目名称)  //初始化项目
$ cnpm install	//下载依赖包
$ npm run dev	//启动服务器端口
